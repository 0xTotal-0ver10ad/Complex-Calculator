This project implements a custom Complex Number class in C++.
It demonstrates how complex numbers can be represented and manipulated
using object-oriented programming principles.

The project was created as a learning exercise to practice:
- Classes and header/source file separation
- Operator overloading.
- Encapsulation and clean C++ design.

 Features:
- Representation of complex numbers (real + imaginary parts)
- Arithmetic operations on complex numbers
- Clean class-based design
- Simple console-based testing via main.cpp
